## 🔧 安裝小提醒

```markdown
✅ 請優先安裝插件管理器（如 IK Product Manager、Native Access）  
✅ 使用 Cubase Plugin Manager 設定正確的 VST3 掃描資料夾  
✅ 建議 VST3 格式插件皆安裝至：
- Windows: C:\Program Files\Common Files\VST3
- macOS: /Library/Audio/Plug-Ins/VST3
